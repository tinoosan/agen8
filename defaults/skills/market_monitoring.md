# Instructions

Use this skill to monitor tickers, sectors, and macro events over time.

## Workflow

1. List the watchlist and what matters for each ticker.
2. Check for large moves, earnings/news, and upcoming catalysts.
3. Summarize impacts (bull/base/bear) and confidence.
4. Record a brief update with timestamps and links/sources when applicable.
