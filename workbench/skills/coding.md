# Instructions

Use this skill for software engineering tasks.

## Workflow

1. Identify the smallest reproducible problem or desired behavior.
2. Locate the relevant code path and constraints.
3. Make focused changes with minimal churn.
4. Add/adjust tests when appropriate.
5. Run the narrowest validation first, then broader checks.
