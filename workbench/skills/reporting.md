# Instructions

Use this skill when the user needs a clear artifact: summary, memo, report, or status update.

## Workflow

1. State the objective and scope.
2. Present findings in bullet points with clear sections.
3. Include decisions, risks, and next steps.
4. Keep it short and action-oriented.
