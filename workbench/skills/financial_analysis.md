# Instructions

Use this skill when you need to evaluate a company’s financial performance or valuation.

## Workflow

1. Define the question (e.g., “Is the stock undervalued?” “Can margins expand?”).
2. Gather inputs (revenue, margins, cash flow, balance sheet items, guidance).
3. Compute key metrics (growth, gross/operating margins, FCF, leverage, ROIC where possible).
4. Form a thesis and list explicit assumptions.
5. Present risks, sensitivity, and what would change your mind.
